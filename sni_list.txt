# sni_list.txt — SNI candidates for ZeroDPI
#
# List one hostname per line.  Lines that start with '#' and blank lines are
# ignored.  ZeroDPI will:
#   1. Resolve each hostname to all its IPv4 addresses via DNS.
#   2. Probe each (hostname, IP) pair: TCP connect → TLS handshake → HTTP GET.
#   3. Rank the results by a composite score (latency, TLS quality, HTTP ok).
#   4. Let you pick one from the interactive table — or auto-select rank-1
#      when AUTO_SELECT = true in config.toml.
#
# Tips:
#   • CDN edge hostnames (Cloudflare, Fastly, Akamai, …) often resolve to
#     many IPs and tend to rank highly because they are fast and reachable.
#   • The scan runs concurrently, so total time is often dominated by the
#     slowest probes, but very large candidate lists may still take longer
#     due to resource limits and scheduler contention.
#   • Hostnames that fail DNS are silently skipped.
